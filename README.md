MinFlux is a minimal, convention based framework for creating React applications
with the Flux architecture.

---

# Installation

```
$ npm install [--save] min-flux
```

---

# Contributing to MinFlux

```
$ git clone git@github.com:jnhuynh/min-flux.git
$ cd min-flux
$ npm install
$ npm install -g gulp
$ npm install -g mocha
```

## Testing MinFlux

For one test run:

```
$ gulp test

or

$ gulp
```

For live reloading:

```
$ gulp live-test
```

# Building MinFlux

```
$ gulp build
```
